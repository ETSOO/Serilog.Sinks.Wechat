# Serilog.Sinks.Wechat
ETSOO's Serilog Sink Package for message publish to Wechat service account.
亿速思维开发的用于消息发布到微信服务帐户的 Serilog Sink。
快速提醒，提高运维反应速度和效率，3分钟提醒一次，避免信息轰炸。
